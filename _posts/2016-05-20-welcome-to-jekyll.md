> git的作用与命令

> - git的作用
>   - 将本地代码推送到远程仓库
>   - 分支管理
> - git命令
>   - git init 初始化本地版本库
>   - git status 查看本地版本库状态
>   - git add . 提交到暂存区（. 表示当前文件中所有内容）
>   - git commit -m "备注" 提交到本地版本库
>   - git clone -b master http:xxx.com 克隆到本地版本库
>   - git push http:xxx.com master:master 从本地版本库推到远程版本库（状态必须为clean）
