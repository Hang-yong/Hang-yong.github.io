## 简介

- 浏览器去加载html文档时，会自上而下一行一行逐行读取
- 浏览器只认识Html文件
- 对于非html的文件，浏览器有能力解析的会展示其源代码，没能力的会自动下载
- html文件语言是超文本标记语言
- 谷歌 火狐 ie safari opera
- 窗口内所包含的是实际大小

## 语法规范

- 用英语单词作为标识，单词两边都有< >
- 一般html标签均是成对出现(开始和结束)
- 因为有一些标签的含义是不限制内容的，这种标签一般会省略结束标签，但是要在开始标签加/
- 标签属性是为了标签展示正确效果提供参数，有属性名就有属性值

```html
<!-- 完整标签 -->
<div>chy<div>
<!-- 单标签 -->
<hr />
```

## 常用标签

- 主体结构

```html
<!-- 告知浏览器本文档采用的是哪个规范 -->
<!DOCTYPE html>
<html>
    <head></head>
    <body></body>
</html>
```

- head标签(头标签)
    - 属性
    - 含义: 定义该页面的信息,行为
    - 子标签
        - title
            - 含义:文档的标题
            - 属性以及属性值
        - meta标签
            - 含义(看效果): 定义该页面的信息 以及 行为
            - 属性以及属性值
                - http-equiv 定义浏览器的什么行为
                    - content-type 编码中文的行为
                    - refresh 刷新页面的行为
                - name 你需要设置的页面信息的名字
                    - keywords---seo优化修改的就是这个设置的具体内容
                - content 所定义的浏览器的行为的具体内容以及所设置的页面信息的具体内容
            - 该标签中的内置api
        - link标签
            - 含义: 引入外部文件
            - 属性以及属性值
                - href 外部文件的地址
                - type 外部文件的类型(MIME类型)
                - rel 外部文件的关系与本文档的关系
                    - stylesheet
                    - icon
- body
    - 属性
    - 含义: 人们俗称的页面
    - 子标签
        - 文本标签
            - br 换行
            - hr 分隔内容
            - p 定义html段落内容
            - h1-6 定义html标题的大小
        - 超链接标签
            -  a标签
                - 含义: 设置超文本链接
                - 属性以及属性值
                    - href 超链接的地址
                    - target 被链接的文档在何处显示
        - 多媒体标签
            - video
                - 含义: 定义视频
                - 属性以及属性值
            - audio
                - 含义: 定义声音
                - 属性以及属性值
                    - src 规定音频文件的 URL
                    - controls 显示音频控件（比如播放/暂停按钮)
                    - width="" height=""
                    - autoplay 自动播放
                    - loop 循环播放
                    - preload 提前加载准备播放，不能与autoplay一起使用
            - img
                - 含义: 插入图像
                - 属性和属性值
                    - src 图像的 URL
                    - width 图像的宽度
                    - height 图像的高度
                    - alt 图片加载不出时代替图片的文本
                    - title 鼠标停留之后出现文字
        - 表格标签
            - table
                - 含义: 定义表格
                - 属性和属性值
                    - border 规定表格单元是否拥有边框
                    - width 表格的宽度
                    - height 表格的高度
                    - align 表格相对周围元素的对齐方式
                        - left center right
                    - cellspacing 单元格之间的空白
                    - cellpadding 单元边沿与其内容之间的空白
            - caption
                - 含义:表格的标题
                - 属性和属性值
                    - align 标题的对齐方式
            - tr
                - 含义:表格的行
                - 属性和属性值
            - th
                - 含义:表格的表头
                - 属性和属性值
            - td
                - 含义:表格单元
                - 属性和属性值
                    - rowspan 跨行合并
                    - colspan 跨列合并
                    - align 单元格内容的水平对齐方式
                        - left right center justify char
        - form表格标签
            - from
                - 含义:创建供用户输入的 HTML 表单
                - 属性和属性值
                    - action 提交表单时向何处发送表单数据
                        - URL
                    - method 规定用于发送表单数据的 HTTP 方法
                        - get
                        - post
            - input
                - 含义:规定了用户可以在其中输入数据的输入字段
                - 属性和属性值
                - name  元素的名称
                - value 指定 input 元素 value 的值
                - type 规定要显示的input元素的类型
                    - text
                    - paszword
                    - radio
                    - checkbox
                    - submit
                - checked 在页面加载时应该被预先选定的元素
                - autofocus 规定当页面加载时 input 元素应该自动获得焦点
                - readonly 规定输入字段是只读的
                - disabled 规定应该禁用的 input 元素
                - required 规定必需在提交表单之前填写输入字段
                - tabindex
            - button
                - 含义:定义一个按钮
                - 属性和属性值
            - textarea
                - 含义:一个多行的文本输入控件
                - 属性和属性值
            - select
                - 含义:创建下拉列表
                - 属性和属性值
            - option
                - 含义:定义下拉列表中的一个选项
                - 属性和属性值
            - label 直接将内容和input表情包含在一块
                - 含义:input 元素定义标注（标记）
                - 属性和属性值
                    - for 规定 label 与哪个表单元素绑定
                    - form 规定 label 字段所属的一个或多个表单
        - 列表标签
            - ul
                - 含义:无序列表
                - 属性和属性值
            - ol
                - 含义:有序列表
                - 属性和属性值
            - li
                - 含义:列表项目
                - 属性和属性值
        - 语意化标签
            - thead
                - 含义:组合 HTML 表格的表头内容
                - 属性和属性值
            - tbody
                - 含义:HTML 表格的主体内容
                - 属性和属性值
            - tfoot
                - 含义:HTML 表格的页脚内容
                - 属性和属性值
            - div
                - 含义:定义 HTML 文档中的一个分隔区块或者一个区域部分
                - 属性和属性值
            - span
                - 含义:对文档中的行内元素进行组合
                - 属性和属性值
            - header
                - 含义:定义文档或者文档的一部分区域的页眉
                - 属性和属性值
            - footer
                - 含义:定义文档或者文档的一部分区域的页脚
                - 属性和属性值
            - section
                - 含义:定义了文档的某个区域
                - 属性和属性值
            - nav
                - 含义:定义导航链接的部分
                - 属性和属性值
